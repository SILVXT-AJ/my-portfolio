# My Portfolio

A modern, responsive, and beautifully designed personal portfolio website built to showcase my projects, skills, and background. 

## 🚀 Tech Stack

This project was built from the ground up using a modern and lightweight tech stack:

* **[React](https://react.dev/) (via Vite)** - For building fast, component-based user interfaces.
* **Vanilla CSS3** - For complete control over the design aesthetic, including modern flexbox/grid layouts, responsive media queries, and smooth micro-animations.
* **JavaScript (ES6+)** - For application logic and interactivity.
* **[React Icons](https://react-icons.github.io/react-icons/)** - For clean, scalable vector SVG icons (Font Awesome icon set) that load instantly without heavy font files.

## ✨ Features

* **Fully Responsive**: Adapts beautifully to mobile, tablet, and desktop screens.
* **Smooth Scrolling Navigation**: A sticky header with smooth-scrolling anchor links for an excellent user experience.
* **Modern Design Aesthetics**: Clean typography, subtle drop shadows, floating hover effects, and a cohesive color palette.
* **Modular Codebase**: Split into logical React components (`Hero`, `About`, `Skills`, `Projects`, `Contact`) for easy maintenance and expandability.

## 🛠️ How to run locally

1. Clone the repository
2. Run `npm install` to download dependencies
3. Run `npm run dev` to start the local development server
